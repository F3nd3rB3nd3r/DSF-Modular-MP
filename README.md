## Still under testing
This mod is still under testing so certain bugs may happen

## Requirements
- https://github.com/ReHamster/DriverNGToolkit/releases
- https://github.com/ReHamster/Alcatraz

## Installation
- Install DriverNG Toolkit
- Install Alcatraz
- Download the latest Modular MP release (https://github.com/F3nd3rB3nd3r/DSF-Modular-MP/releases --> zip file)
- Open the **Modular_MP.zip** file
- Paste the **Common** folder into your Driver:San Francisco installation path
- Enjoy
